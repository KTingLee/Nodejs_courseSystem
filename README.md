## 技術(使用到的模組)：
1. Node.js
2. Express 框架：建立路由(RESTful 風格)
3. express-session：登入成功後，要紀錄 session
4. cookie-parser：登入成功後，傳遞 cookie
5. ejs 模板：網頁模板
6. formidable：處理 post 表單
7. mongoDB (mongoose)：記錄學生資訊

除了上述的模組外，還會應用 jQuery 函式庫、Ajax 發送請求、Bootstrap 樣板。
